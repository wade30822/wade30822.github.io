阻塞队列是一种很特殊的队列，当线程尝试从一个空的阻塞队列中获取数据或者向一个满的阻塞队列中放入数据时都会阻塞。线程从一个空阻塞队列中获取数据将阻塞直到其他线程向其中插入了数据，而线程向一个满的阻塞队列中放入数据也将阻塞直到其他线程从该队列中取出数据。
